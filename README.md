**Command line utility for downloading pages from Internet.**
###Usage:    
page-loader [-h] [-o OUTPUT] url

positional arguments:
  url

optional arguments:
  -h, --help            show this help message and exit
  -o OUTPUT, --output OUTPUT
                        Specify a directory for loading
